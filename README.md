## üöÄ Getting Started

### Prerequisites

To view and edit these diagrams, you'll need:

- [Mermaid Live Editor](https://mermaid.live/) (for viewing and editing online)
- [VS Code](https://code.visualstudio.com/) with the [Mermaid extension](https://marketplace.visualstudio.com/items?itemName=bierner.markdown-mermaid) (for local editing)
- [Mermaid CLI](https://github.com/mermaid-js/mermaid-cli) (for generating images from .mermaid files)

### Viewing the Diagrams

1. Clone this repository
2. Open any `.mermaid` file in the `codes/` directory
3. Copy the content to [Mermaid Live Editor](https://mermaid.live/) or use a supported editor

## üõ† Usage

To edit or create new diagrams:

1. Install the Mermaid extension for your preferred editor
2. Create a new `.mermaid` file or edit an existing one
3. Use Mermaid syntax to create your diagram
4. Save your changes and commit them to the repository
